Staging Slot

App Service Republication Lab

This sample web app was deployed to a staging deployment slot as part of the AZ-104 hands-on lab.

The deployment used the following Azure PowerShell command sequence:

  1. New-AzResourceGroup
  2. New-AzAppServicePlan
  3. New-AzWebAppSlot
  4. Publish-AzWebApp

Test users can review this page before the app is swapped to production.
