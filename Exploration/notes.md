  - Azure DNS
  - Azure Bastion
  - Azure Route Server
  - NAT Gateway
  - Traffic Manager
  - Load Balancer
  - Application Gateway
  - Azure Front Door
  - VPN Gateway
  - ExpressRoute
  - Virtual WAN
  - Peering
  - Azure Firewall
  - Web Application Firewall (WAF)
  - DDoS Protection
  - Azure Monitor
  - Azure Virtual Network Manager
  - Container Networking Services
