Coverage notes:

* This task is fragmented across several documentation areas. Azure Migrate and Cloud Adoption Framework are the primary sources, while Azure Architecture Center and Well-Architected guidance help with service-selection tradeoffs.
* Architecture guidance is more useful than a single product root because the task asks for a recommendation between IaaS and PaaS migration paths, not just tool configuration.
* Download first: Azure Migrate, Cloud Adoption Framework, Azure Architecture Center, Virtual machines in Azure, and Azure App Service.
* Major products that could be testable but are only lightly covered here: Azure Container Apps, AKS, Azure VMware Solution, Azure Arc-based discovery, Azure Site Recovery, Private Link, VPN Gateway, ExpressRoute, Azure Monitor, and Azure Advisor.
